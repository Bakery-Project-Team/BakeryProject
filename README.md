# BakeryProject

The Delivery Management System (DMS) is a mobile application designed to streamline wholesale delivery processes for Zephirin’s Bakeries. Drivers can efficiently manage assigned deliveries, download and print invoices, log returns or discrepancies all within a user-friendly interface. This system addresses the manual, error-prone processes previously used, offering improved accuracy and reduced delays.

Built with Ionic, Angular, and SQLite, the DMS ensures offline functionality for regions with limited connectivity and is optimized for mobile devices to accommodate drivers with minimal technical experience.

How to Open
Ensure you are using a POS terminal device running Android for the best experience (OCOM POS T2Pro Handheld Terminal).

Download the DMS.apk file onto the POS terminal device.

Locate the APK file on the device and install it (ensure permissions for unknown sources are enabled).

Open the installed application from the app drawer.
